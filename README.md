# Application de gestion de contacts

## Description du projet
Cette application permet aux utilisateurs d'ajouter leurs contacts, de les mettre à jour et de les supprimer aprés s'être connecté.

## Fonctionnalités

### Phase 1 – Gestion des contacts
- Affichage de vos contacts.
- Création d'un nouveau contact.
- Mise à jour d'un contact.
- Suppression d'un contact.

### Phase 2 – Gestion des messages
- Possibilité d’envoyer des messages à un de nos contacts.
- Possibilité de recevoir des messages de nos contacts.
